# دفتر المتجر — إدارة مكتبة وإعلام آلي

إضافة كروم لإدارة مخزون ومبيعات وديون متجر (كتب، أدوات مكتبية، معلوماتية).

## المزايا
- **المخزون**: إضافة/تعديل/حذف منتجات، سعر شراء وبيع، تنبيه عند انخفاض الكمية.
- **المبيعات**: نقطة بيع بسيطة (سلة + وصل)، خصم تلقائي من المخزون، دفع نقدي أو دين.
- **الديون**: بطاقة لكل زبون، تسجيل التسديدات الجزئية والكاملة.
- **التقارير**: مبيعات وأرباح اليوم والشهر، الأكثر مبيعًا، المخزون المنخفض، إجمالي الديون.

كل البيانات تُحفظ محليًا في متصفحك (`chrome.storage.local`) ولا تُرسل لأي خادم خارجي.

## الرفع على GitHub
1. أنشئ مستودعًا (repository) جديدًا على GitHub.
2. فك ضغط هذا الملف، وارفع كل محتوياته (manifest.json, index.html, ...) إلى المستودع.

## تحميل الإضافة في Chrome
1. افتح `chrome://extensions` في المتصفح.
2. فعّل **وضع المطوّر** (Developer mode) من الزاوية العلوية.
3. اضغط **تحميل غير مضغوط** (Load unpacked).
4. اختر المجلد الذي يحتوي على `manifest.json` (مجلد الإضافة بعد فك الضغط أو بعد تنزيله من GitHub).
5. ستظهر أيقونة "دفتر المتجر" بجانب شريط العنوان — اضغط عليها لفتح التطبيق في تبويب جديد.

## ملاحظة حول التحديث من GitHub
تحميل الإضافة بهذه الطريقة (Unpacked) لا يتحدّث تلقائيًا. عند تعديل الملفات على GitHub، نزّل النسخة الجديدة يدويًا واضغط زر **تحديث (⟳)** في صفحة `chrome://extensions`.
